## Install InstantSearch

InstantSearch is an open source Javascript library that lets you quickly build fast and delightful search interfaces for your front-end application.

Add our InstantSearch `npm` package to your project
<CodeGroup>
  ```bash Javascript
  npm install algoliasearch instantsearch.js instantsearch.css
  ```
  ```bash React
  npm install algoliasearch react-instantsearch instantsearch.css
  ```
  ``` bash Vue
  npm install vue-instantsearch algoliasearch instantsearch.css
  ```
</CodeGroup>
